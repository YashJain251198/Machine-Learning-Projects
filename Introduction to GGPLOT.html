Create a Scatterplot between 'mpg' and 'hp'¶

Now, GGPLOT is package that runs in layers. Lets Look into the syntax to understand it better

ggplot(dataset, aes(x=, y=)) + geom_point

Lets Understand the Syntax. There are 03 layers in the code above. Lets understand it in simple language.

ggplot - it is the name of the function without which the plot will not be created.

dataset - Name of the data from which the plot will be created. This is the First Layer.

aes - known as Aesthetics. This layer is used to define the X-Axis and Y-Axis of the Plot. (Here in above e.g. we will write x=mpg, y = hp). This is Second Layer.

Note - In case you are a new user and using R for the first time, please note that R is case sensitive which means that variables name should match with MTCARS Dataset.

geom_point() - This command is required to create a scatterplot in R. This is the third layer.

Now, lets write the code and run it.

In [10]:
ggplot(mtcars,aes(x = mpg, y = hp))+geom_point()

Here mpg is Miles Per Gallon and hp is Horsepower

Lets change the colors of the datapoints of scatterplot created above¶

In order to change the color of the scatterplot data points, we need to use color = "red" in geom_point.

Now one can define any color of their choice instead of red. Lets write the code and see the result.

In [11]:
ggplot(mtcars,aes(x = mpg, y = hp))+geom_point(color = "red")

Can We Change the Size of the Datapoints...Lets See if this can be done...¶

Now just like the way we have changed the color of the datapoints, similarly we can use geom_point(size = 2.5)

Tip - Try to keep the points between 1 to 5 unless very large datapoints are required.

In [12]:
ggplot(mtcars,aes(x = mpg, y = hp))+geom_point(color = "red", size = 2.5)

Shape and Size of Data Points in Scatterplot¶

Ok, so now we have seen that we can change the color of the datapoints. Now,can we change the size and shape of the datapoints. Lets see if this can be done.

In [13]:
ggplot(mtcars,aes(x = mpg, y = hp))+geom_point(color = "red", size = 2.5, shape = 1)

The point worth noting in above plot is as we passed the parameter "shape = 1", the datapoints shape converted from opaque dots to hollow circles. There are lots of shapes and codes available in the google drive which is available under the following link:

Background of the Plot¶

If you wish to change the background of the plot, we can use theme_classic() to do the job.

In [15]:
ggplot(mtcars,aes(x = mpg, y = hp))+geom_point(color = "red", size = 2.5, shape = 1)+theme_classic()

Lets Put Labels on the Scatterplot¶

In [17]:
ggplot(mtcars,aes(x = mpg, y = hp))+geom_point(color = "red", size = 2.5)+theme_classic()+
labs(title = "Scatterplot - MPG Vs HP",
    subtitle = "Scatterplot showing Negative Correlation",
    x = "Miles Per Gallon",
    y = "Horsepower",
    caption = "Source: MTCARS Dataset")

Here Labs is a command that adds labels to the plot. Referring to the above plot

title - Main Title of the Plot

subtitle - Text Below the Main Title

x - X Axis Label

y - Y Axis Label

caption - Providing Info about the Plot. Shown in the Bottom Right Corner of the Plot

Lets Put the Color on the Plot basis the Column Values¶

In order to color the plot basis column values, we have to define the column name is Aesthetics. Lets See how this can be done.

MTCARS Data - Color the Datapoints basis Cylinder Column Values.¶

In [19]:
ggplot(mtcars,aes(x = mpg, y = hp))+
geom_point(aes(color = factor(cyl)), size = 3.5)+
theme_classic()+ labs(title = "Scatterplot - MPG Vs HP",
    subtitle = "Scatterplot showing Negative Correlation",
    x = "Miles Per Gallon",
    y = "Horsepower",
    caption = "Source: MTCARS Dataset")

It is visible that by putting color = factor(cyl) in aesthetics column under geom_point() changes the entire color spread of the plot. It can be clearly visible by the color that 4 cylinder car with horsepower upto 130 approx has a fuel efficiency of 20 - 35 kmpl.

Moreover, factor(cyl) is taken because cylinder column has only 03 types of values namely 4, 6 and 8 cylinders.

Similarly, we can change the shape and size of the datapoints in the scatterplot. Lets see how this can done....¶

In [20]:
ggplot(mtcars,aes(x = mpg, y = hp))+
geom_point(aes(color=factor(cyl), shape = factor(cyl),size = factor(cyl)))+
theme_classic()+ labs(title = "Scatterplot - MPG Vs HP",
    subtitle = "Scatterplot showing Negative Correlation",
    x = "Miles Per Gallon",
    y = "Horsepower",
    caption = "Source: MTCARS Dataset")

Warning message:
“Using size for a discrete variable is not advised.”

Thus, it can be seen that using aesthetics and defining the color, shape and size basis column values change the entire appearance of the plot and becomes easy to interpret.
